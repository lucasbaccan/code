
# Code 🛠

## Introdução

Este projeto tem o intuito de agregar a maior quantidade de conteúdos de forma simples e eficiente, que auxilie no dia a dia.

Seu desenvolvimento ocorreu utilizando *Docusaurus 2*, o qual pode ser visto mais informações nos seguintes links:

-  [Site oficial](https://v2.docusaurus.io/)
-  [Docusaurus.md](Docusaurus.md) 

## Rodar

Para rodar o projeto, rode o comando:
```
$ npm start
```
## Construir
Para construir o *build* do projeto, rode o comando:
```
$ npm run build
```
O site vai ser gerado na pasta `./build`
